Projects

Leaf dryer oven

Here I made, with the help of a social service co-worker, a Leaf Dryer Oven controlled with an Android app using Bluetooth communication on a Master/slave model in which the app was the Master and an Arduino was the slave that put in motion the oven and read the temperature

Geolocation app for area reports

In this app I made an Android app that used the Google Maps API to retrieve the geolocation data and then allow the user to fill in a report that included photos of the problem (by which I used Read/Write permissions for data storage and the Camera X library to allow the camera use in-app).

Also, I used Room Persistence Library that functions with SQLite queries to store internally the data of the reports. And in addition to this I implemented Firebase for Authentication purposes.

Barcode reader app

This app consisted mainly on a barcode reader that decoded the data inside an internal barcode standard used for the company as identifiers in certain products and also served to track the user's productivity with the readings in a labor session. I achieved this by using the ML Kit by Google and the previously mentioned Camera X Library with some knowledge I previously learned, like Room Persistence Library to store the data decoded.

At the end the decoded lectures could be sent to the company's servers using the FTP Protocol.
